GameCube & Wii Support

Thanks to the GameCube Loader plugin, Ghidra has gained access to GameCube and Wii executables and a couple of related features:

  • Executable support: .dol, apploader
  • Library support: .rel, .rso
  • MEM1 RAM dump
  • Yaz0 compressed files (as a file system)
  • Symbol map loading and exporting
  • CodeWarrior Demangler
  • TOC (SDA, SDA2) and GQRn analyzer for working Small DAta offsets and values
  • PowerPC Gekko Processor support

To get started, simply import a .dol file. You can get one by letting Dolphin Emulator extract system files from the disc.
